# Roadmap

Planned features and improvements for Bullhorn.

## In Progress

- **Auto-publishing** — Automatically publish scheduled posts to Twitter, LinkedIn, and Reddit at their scheduled time via platform APIs
- **Stripe billing** — Free and paid tiers with usage-based limits on posts, storage, and API calls

## Planned

- **Analytics dashboards** — Track post performance (impressions, clicks, engagement) with connected platform accounts
- **Team collaboration** — Shared workspaces with role-based access for teams managing social content together
- **More platforms** — Bluesky, Mastodon, and Threads support
- **Post templates** — Reusable templates for recurring content formats
- **Bulk scheduling** — CSV/spreadsheet import for scheduling many posts at once
- **AI drafting** — AI-assisted post writing with platform-specific tone and formatting suggestions

## Completed

- Multi-platform drafting (Twitter, LinkedIn, Reddit)
- Campaigns and projects organization
- Blog drafts with Markdown
- Launch posts workspace
- Media uploads via Supabase Storage
- API keys and MCP server
- iOS app (Capacitor)
- Web push and iOS push notifications
- Community events calendar
- Google OAuth and email/password auth
- Rate limiting (Upstash Redis)
- Error monitoring (Sentry)
